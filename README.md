# Projects_Master_TSE

Academic projects completed during my studies at the Toulouse School of Economics.

This repository gathers a selection of projects completed across statistics, econometrics, data analysis, machine learning, databases, network analysis, spatial methods, and actuarial modeling.

## Projects

### COVID Database Design and SQL Analysis
Relational database design project based on COVID-19 data, combining entity-relationship modeling and SQL-based analytical queries on vaccination, fatality rates, demographics, and healthcare capacity.

### Data Mining – GGM and CGGM
Project on Gaussian Graphical Models (GGM) and Clusterpath Gaussian Graphical Models (CGGM) applied to macroeconomic data, comparing industrialized countries and natural-resource-rich countries.

### Extreme Value Analysis of the USD/EUR Exchange Rate
Project using Extreme Value Theory (EVT) to model the heavy tails of USD/EUR exchange rate returns, with applications to tail estimation, GEV/GPD modeling, Value at Risk, and Expected Shortfall.

### Geomarketing Retail Location Analysis
Retail location analysis project combining gravity models, principal trading areas, and spatial econometrics to identify promising locations for future store openings. The project also includes an interactive Shiny application.

### Spatial Cluster Detection in Boston Housing Data
Project on spatial scan statistics applied to Boston housing data, combining cluster detection, residual analysis, and spatial econometric modeling to identify local spatial patterns not captured by standard regression approaches.

### Spurs Assist Network Analysis
Network analysis project on the San Antonio Spurs’ assist network during the 2024–25 NBA regular season, studying whether player popularity is associated with assist tie formation and offensive centrality.

### Survey Sampling – IPPS Designs
Project on inclusion probability proportional-to-size sampling designs, comparing IPPS with replacement, Poisson IPPS without replacement, and Rao-Sampford sampling through Monte Carlo simulations.

### Survival Analysis – Parametric Models
Project on parametric survival models, with a focus on AFT specifications, distribution validation, model comparison, and applications to life insurance contract durations.

## Notes

- Some projects were completed individually, while others were carried out in groups.
- Each folder contains the main project materials available, such as reports, slides, code, datasets, or visual outputs.
- This repository is intended to provide a structured overview of my academic work at TSE.
