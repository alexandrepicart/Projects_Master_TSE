# Macroeconomic Structures with GGM and CGGM

This project studies the macroeconomic structures of industrialized countries and natural-resource-rich countries using Gaussian Graphical Models (GGM) and Clusterpath Gaussian Graphical Models (CGGM).

The objective was to identify direct conditional relationships between macroeconomic variables and compare how these dependency structures differ across groups of countries.

## Project Contents

- Presentation slides
- R Markdown code
- Macroeconomic network analysis
- Comparison of industrialized and resource-rich countries

## Files

- `Project_Slides.pdf`: presentation slides explaining the motivation, data, methodology, and main empirical results.
- `Project_Code.Rmd`: R Markdown file containing the code used for data preparation, HAR preprocessing, and GGM/CGGM estimation.

## Data

The analysis relies on World Development Indicators (WDI) data from the World Bank over the 2005–2021 period.

The final sample includes 19 countries, 10 continuous macroeconomic variables, and 391 observations.

## Main Methods

- Gaussian Graphical Models (GGM)
- Graphical lasso
- Clusterpath Gaussian Graphical Models (CGGM)
- HAR preprocessing
- Dendrogram and cluster interpretation

## Main Objective

The project compares the conditional dependency structures of macroeconomic variables across two groups of countries:
- industrialized countries
- natural-resource-rich countries

It also includes a specific comparison for Norway, whose position is intermediate between the two groups.
