# Extreme Value Analysis of the USD/EUR Exchange Rate

This project was completed as part of the Extreme Value Analysis course at the Toulouse School of Economics.

The objective was to study the extreme behaviour of the USD/EUR exchange rate using Extreme Value Theory (EVT). The analysis was based on daily exchange rate data from January 2004 to December 2023 and focused on modeling heavy tails, estimating tail indices, and deriving risk measures such as Value at Risk (VaR) and Expected Shortfall (ES).

## Project Contents

- Final report
- R code
- Empirical analysis of heavy tails
- EVT modeling and risk metrics

## Files

- `Projet_AVE_Groupe4.pdf`: final written report presenting the data preparation, EVT methodology, model estimation, diagnostics, and main results.
- `Code_AVE_Project.R`: main R script used for the empirical analysis, including data cleaning, log-return transformation, tail estimation, GEV/GPD modeling, and risk measure computation.

## Main Methods

- Log-return transformation
- Tail index estimation
- Hill and moment estimators
- Block Maxima (GEV)
- Peaks Over Threshold (GPD)
- Value at Risk (VaR)
- Expected Shortfall (ES)
