# Spatial Cluster Detection in Boston Housing Data

This project was completed at the Toulouse School of Economics and focuses on spatial cluster detection methods applied to housing data.

The objective was to detect spatial clusters in the Boston housing dataset using scan statistics, and to study whether these clusters could help explain residual spatial structure left by econometric models.

## Project Contents

- Final report
- Presentation slides
- R Markdown code
- Spatial econometric analysis

## Files

- `Projet_cluster_detection.pdf`: final written report presenting the methodology, empirical application, and main findings.
- `Slides_cluster_detection.pdf`: presentation slides summarizing the motivation, scan test method, empirical application, and model comparison.
- `Projet_cluster_detection.Rmd`: R Markdown file containing the code used for the analysis, including data preparation, scan test implementation, residual analysis, and spatial econometric modeling.

## Main Methods

- Spatial scan test
- Monte Carlo permutation procedure
- OLS regression
- Spatial Error Model (SEM)
- Residual spatial cluster detection
- Moran’s I
- Spatial data visualization in R

## Main Objective

The project aimed to identify spatial clusters in housing-related variables and in model residuals, in order to better understand local spatial heterogeneity and improve econometric specification.
