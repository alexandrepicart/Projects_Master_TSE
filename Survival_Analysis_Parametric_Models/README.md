# Survival Analysis – Parametric Models

This project was completed as part of a Survival Analysis course at the Toulouse School of Economics.

It focuses on parametric survival models, their interpretation, and their validation, with an application to life insurance contract durations.

## Project Contents

- Presentation slides
- Parametric survival modeling
- Model validation and comparison

## Files

- `Survival_Project_Slides.pdf`: presentation slides explaining the main parametric survival distributions, AFT models, model validation tools, and the empirical application.

## Main Methods

- Exponential model
- Weibull model
- Log-normal model
- Log-logistic model
- Gompertz model
- Accelerated Failure Time (AFT) models
- AIC / BIC model comparison
- Cox-Snell residuals

## Application

The empirical illustration is based on life insurance contracts and studies contract duration with right-censored data.
