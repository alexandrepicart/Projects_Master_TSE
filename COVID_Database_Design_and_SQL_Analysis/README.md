# COVID Database Design and SQL Analysis

This project was completed as part of a Databases course.

The objective was to design a relational database for COVID-19 data and use SQL-based analysis to investigate several questions related to pandemic dynamics, vaccination, demographics, and healthcare capacity.

## Project Contents

- Database design
- Entity-relationship model
- Analytical query design
- COVID-19 cross-country analysis

## Files

- `Databases_Report.pdf`: final written report presenting the database structure, entity-relationship diagram, data import process, and the main analytical queries and results.

## Main Components

- Country dimension
- Time dimension
- COVID statistics fact table
- Vaccination statistics fact table
- Country profile table
- Country age structure table

## Main Analyses

The project explores several questions related to the COVID-19 pandemic, including:

- the relationship between GDP per capita and fatality rates
- seasonal variation in COVID-19 mortality
- temperature and infection intensity
- vaccination progress and case fatality rates
- elderly population share and fatality rates
- healthcare staff density and mortality
- pandemic severity over time in France
